# AI Web Studio Pro 2026

## 概要・説明
AI Web Studio Pro 2026は、ブラウザ上でAIと対話しながらWebアプリケーションを設計・実装し、GitHubへのデプロイまでをシームレスに行える次世代のAI駆動Web開発プラットフォームです。

ユーザーはチャットインターフェースを通して要件を伝え（画像添付も可能）、Geminiモデルを活用して完全なHTML/CSS/JSコードを自動生成します。内蔵のエディタとリアルタイムプレビュー機能を搭載し、修正作業も直感的に行えます。さらに、Service Workerを活用したバックグラウンド処理により、タブを閉じてもコード生成が継続します。完成したコードはGitHubリポジトリへ直接コミットし、GitHub Pagesによる自動公開までワンストップで完結します。

## 主な機能一覧

- **AIアシスタント機能 (PLAN / BUILD)**
  - Gemini APIと連携し、対話形式でアプリケーションの要件定義と設計をサポート。
  - 画像の添付機能（カメラ、ギャラリー、ファイル）によるマルチモーダルな指示。
  - 複数モデルの切り替え対応（2.5 Flash, 2.5 Pro, Deep Research等）。
- **強力なコードエディタ＆プレビュー**
  - 生成されたソースコードを内蔵エディタで直接編集可能。
  - iframeを利用した即時プレビュー機能。
  - プレビュー画面から気になった部分を直接AIに「修正依頼」できるデバッグパネル機能。
- **バックグラウンド生成**
  - Service WorkerとIndexedDBを利用し、時間のかかるコード構築中もバックグラウンドで処理を継続。
- **高度なGitHub連携**
  - **Import**: GitHub上の既存リポジトリからコードを直接読み込み。
  - **Deploy**: 新規リポジトリの作成、自動生成されたREADME.mdとソースコードのコミット。
  - **GitHub Pages**: ワンクリックでGitHub Pagesを有効化し、生成したWebアプリを即座にインターネットへ公開。
- **ユーザーフレンドリーなUI/UX**
  - モバイルファースト設計、ダークモード基調のモダンなUI。
  - スワイプ操作によるタブ切り替え機能（CHAT / CODE / PREVIEW / SET）。

## 使用技術・ライブラリ

本プロジェクトは外部ライブラリ（ReactやVueなど）に依存せず、モダンなWeb標準技術のみで構築されています。

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript
- **永続化・オフライン処理**: IndexedDB, Service Worker
- **外部API連携**:
  - Google Gemini API (AIチャット、ソースコード生成、README自動生成)
  - GitHub REST API (リポジトリ管理、ファイル取得、コミット作成、Pages設定)
- **バックエンド/プロキシ**: Cloudflare Workers（APIキー未設定時のプロキシとして利用）

## セットアップ・使い方

### 1. 起動方法
1. 本リポジトリの `index.html` をダウンロードします。
2. ブラウザでそのままファイルを開くか、任意のローカルサーバー環境（Live Serverなど）でホストしてアクセスします。

### 2. 初期設定 (SETタブ)
1. 画面右上の **SET (設定タブ)** を開きます。
2. **API KEY**: お持ちのGemini APIキーを入力します（未入力の場合は内蔵のプロキシワーカーが使用されますが、制限がある場合があります）。
3. **GITHUB TOKEN**: GitHub Deploy機能を利用する場合は、`repo` スコープ権限を持ったGitHub Personal Access Tokenを入力します。
4. **DEVELOPER PROFILE**: 「Tailwind CSSを使いたい」「コメントは日本語で」など、AIに守らせたいコーディングスタイルや好みを設定して保存します。

### 3. アプリケーションの開発 (CHAT & PREVIEWタブ)
1. **CHATタブ** で、「ダークモードのTODOアプリを作って」のようにAIに要件を伝えます。必要に応じてUIのモックアップ画像を添付することもできます。
2. AIと仕様をすり合わせ、プランが確定すると **「⚡️ コードを作成 (BUILD)」** ボタンが表示されるのでクリックします。
3. コード生成が完了したら、**PREVIEWタブ** で実際の動作を確認します。
4. 修正したい場合は、PREVIEW画面下部の「修正を依頼」から追加の指示を送ることができます。また、**CODEタブ** で直接コードを手直しすることも可能です。

### 4. GitHubへの公開・デプロイ
1. アプリが完成したら、**CODEタブ** 右上の **「🚀 GitHub Deploy」** をクリックします。
2. デプロイ先のリポジトリを選択するか、「＋ 新規リポジトリを作成」から新しいプロジェクトを立ち上げます。
3. デプロイ画面では、AIが現在のコードを読み取って自動作成した `README.md` の内容を確認・編集できます。
4. 「GitHub Pagesを有効にして公開する」にチェックを入れた状態で **「🚀 デプロイ実行」** をクリックします。
5. 処理が完了すると、公開されたサイトのURLとリポジトリへのリンクが表示されます。
