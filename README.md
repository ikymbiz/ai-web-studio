# 🌌 AI Web Studio Pro 2026

AIとの対話（チャット）を通じて、Webアプリケーションの要件定義からコード生成、プレビュー、そしてGitHubへのデプロイまでを**ブラウザ上の単一画面で完結**させる、次世代のオールインワン開発ツールです。

洗練されたダークUI（Obsidian Terminal Design）と、モバイル・PC両対応のレスポンシブ設計を備えています。

## ✨ 主な機能 (Features)

### 💬 1. 対話型 AI 開発アシスタント (CHAT)
*   **プランニング＆ビルド**: まずAIとチャットで要件定義（Plan）を行い、仕様が固まったらコードを生成（Build）する堅実な開発フロー。
*   **マルチモーダル対応**: カメラ撮影、ギャラリー、ファイルから画像を添付し、UIのモックアップ画像などからAIにデザインを指示可能。
*   **スマートな履歴管理**: チャット履歴のローカル保存、メッセージの編集・削除機能。

### 💻 2. 高機能コードエディタ (CODE)
*   **CodeMirror統合**: シンタックスハイライト、スムーズなスクロール、カスタム検索機能を備えた本格的なエディタ環境。
*   **ファイル操作**: ローカルからのコードアップロード、PCへのダウンロード。
*   **GitHubインポート**: 既存のGitHubリポジトリから対象のコードを直接読み込み。

### 👁️ 3. リアルタイムプレビュー＆デバッグ (PREVIEW)
*   生成・編集したコードを隣接するiframeで瞬時にプレビュー。
*   **デバッグアシスト**: プレビュー画面を見ながら「修正したい点」をメモ欄に入力するだけで、AIが文脈を理解し、修正案・デバッグコードを提案。

### 🚀 4. GitHub ダイレクトデプロイ (DEPLOY)
*   アプリ内から直接新規リポジトリを作成、または既存リポジトリを選択。
*   デプロイ実行と同時に**自動的にREADMEをAIが生成**。
*   ワンクリックで **GitHub Pages** を有効化し、作成したWebアプリを世界中に即座に公開。

### ⚙️ 5. バックグラウンド生成 & ローカル保存
*   **Service Worker**: 重いコード生成タスクをバックグラウンドで実行。タブを移動しても処理が中断されません。
*   **IndexedDB連携**: チャット履歴、現在のコード、APIキーなどの設定情報はすべてブラウザのローカルデータベースに安全に自動保存されます。

## 🛠 使用技術 (Tech Stack)

*   **Frontend**: HTML5, CSS3, Vanilla JavaScript (フレームワークレスの単一ファイル構成)
*   **Editor**: [CodeMirror 6](https://codemirror.net/) (ESM経由で動的インポート)
*   **AI API**: [Google Gemini API](https://ai.google.dev/) (2.5 Flash, 2.5 Pro, 3.0 Flash 等に対応)
*   **Storage**: IndexedDB (APIキー、コード、履歴の永続化)
*   **Worker**: Service Worker (バックグラウンド処理)
*   **Integration**: GitHub REST API

## 🚀 使い方 (Getting Started)

本アプリケーションは、ビルド不要のシングルファイル構成です。

1. **起動する**
   * 本ファイル（`index.html`）をダウンロードし、最新のWebブラウザ（Chrome, Edge, Safari推奨）で開きます。
2. **初期設定をする (SET タブ)**
   * `API KEY`: Google Gemini APIキーを入力します。（※未設定の場合は、デモ用のプロキシワーカーを経由して簡易モデルが動作します）
   * `GITHUB TOKEN`: デプロイ機能を使用する場合は、`repo` スコープを持つ GitHub Personal Access Token を設定してください。
   * `DEVELOPER PROFILE`: AIに守らせたいコーディングスタイル（例: "Tailwind CSSを使用", "コメントは日本語"など）を記述します。
3. **アプリを作る (CHAT タブ)**
   * 「ダークモードのTodoアプリを作りたい」など、要件をAIに伝えます。
   * AIから提案されるプランに合意すると「⚡️コードを作成 (BUILD)」ボタンが出現します。
4. **プレビューと調整 (PREVIEW / CODE タブ)**
   * プレビュー画面で動作を確認し、必要に応じてエディタで微調整、またはデバッグパネルからAIに修正を依頼します。
5. **公開する (CODE タブ -> 🚀 GitHub Deploy)**
   * 「🚀 GitHub Deploy」ボタンを押し、リポジトリ名等を指定してデプロイします。AIがREADMEも自動で書き上げます。

## 📱 モバイル対応について

本ツールはスマートフォンやタブレットでの利用にも最適化されています。
*   **スワイプナビゲーション**: 画面を左右にスワイプすることで、CHAT → CODE → PREVIEW のタブをシームレスに切り替えられます。
*   **カメラインテグレーション**: モバイル端末のカメラを直接起動し、手書きのワイヤーフレームを即座にAIに読み込ませることが可能です。

## 🔐 セキュリティとプライバシー

*   **APIキーとトークンの保護**: 入力された Gemini API キー および GitHub トークン は、お使いのブラウザのローカルストレージ（IndexedDB）にのみ保存されます。外部のサーバーに送信・蓄積されることはありません。
*   **コード履歴のプライバシー**: 開発中のコードや会話履歴もすべてローカル内で完結します。
