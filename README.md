# 🪐 AI Web Studio

> ブラウザで完結する、AI駆動型のオールインワンWebフロントエンド開発スタジオ

AI Web Studioは、自然言語による指示からHTML/CSS/JSコードを生成し、ブラウザ上で直接編集・プレビュー・デプロイまでをシームレスに行うことができる画期的なWeb開発環境です。
ChatGPT、Gemini、Claudeなどの最先端LLMを切り替えて使用でき、チャット形式で要件定義を進めながら、ワンストップでWebサイトを構築できます。サーバーの構築は不要で、シングルHTMLファイルとして動作します。

---

## ✨ 主な機能

### 💬 AIチャット ＆ 自動要件定義
- **マルチモーダル入力**: テキストだけでなく、カメラ撮影、デバイス内の画像、あらゆるファイルを添付してAIに指示が可能。
- **仕様の自動抽出**: AIとの会話から「実装すべき要件」を自動で抽出し、リスト化。手動での追加やスワイプ操作での編集・削除（モバイル対応）も可能です。
- **ユーザーペルソナ学習**: 会話内容からユーザーの開発の好み（使用技術、デザインテイスト）をAIが自動抽出し、プロファイルを更新・提案します。
- **マルチプロバイダー対応**: Gemini, OpenAI, Claude, xAI, Groqに対応。用途に合わせて最適なモデルを選択できます。

### 💻 高機能コードエディタ
- **CodeMirror 6 統合**: シンタックスハイライト、コード内検索機能（Find in Code）を備えた本格的なエディタを内蔵。
- **GitHub Import**: 既存のGitHubリポジトリを検索し、ファイルを直接インポートして編集できます。
- **ローカルファイル連携**: デバイスからのコードアップロードや、完成したコードのダウンロード（`index.html`）に対応。

### 👁️ リアルタイムプレビュー ＆ デバッグ
- **即時プレビュー**: エディタでの変更が即座に反映されるプレビュー用iframeを搭載。
- **ダイレクト修正依頼**: プレビューを見ながら「ボタンの色を赤にして」などとメモを入力するだけで、AIが原因を特定しコードを修正します。

### 🚀 GitHub Pages ワンクリックデプロイ
- **自動デプロイ**: 作成したコードをGitHubの新規・既存リポジトリへ直接プッシュ。
- **ホスティング連携**: デプロイと同時にGitHub Pagesを有効化し、公開完了までのカウントダウンとURLの発行を行います。

### 🎨 柔軟なカスタマイズとアセット管理
- **デザインテーマ機能**: GUIで配色テーマを作成し、生成されるコードへCSS変数として自動的に注入します。
- **テンプレート管理**: よく使うHTMLボイラープレートをカスタムテンプレートとして保存し、ワンクリックで呼び出し可能。
- **プロンプトチューニング**: システムプロンプトやコーディング原則（Code Principles）、各アクション時のプロンプトを細かく調整可能です。

## 🔐 セキュリティ・プライバシー

- **APIキーのローカル暗号化**: 登録されたAPIキーやGitHubトークンは、`Web Crypto API` を用いて強力に暗号化（AES-GCM/PBKDF2）され、ブラウザの `IndexedDB` にローカル保存されます。
- **自動消去システム**: 保存されたAPIキーは2週間経過すると自動的に破棄される安心設計。
- **セキュアな通信**: 外部のデータベース等を経由せず、通信は直接各APIプロバイダーに対して行われます。

## 🛠️ 使い方

### 1. 初期セットアップ
1. 本HTMLファイルをブラウザで開きます。スマートフォンからのアクセス（PWA対応）にも最適化されています。
2. 下部のタブから **`SET` (設定)** を開きます。
3. 使用したいAIのプロバイダーを選択し、該当の **APIキー** を入力します。（無料枠の大きい Gemini API の利用を推奨します）
4. （オプション）GitHubデプロイ機能を使用する場合は、**GitHub Token** (`ghp_...`) を設定してください。

### 2. 基本ワークフロー
1. **要件定義 (CHAT)**: チャット画面で「ダークモード対応のTodoアプリを作りたい」と送信します。AIが内容を分析し、右上の「📋 仕様」バッジに要件がストックされていきます。
2. **生成 (BUILD)**: 要件が固まるとチャット上に「⚡️ コードを作成 (BUILD)」ボタンが出現します。クリックするとAIが一気にコードを記述します。
3. **確認 (PREVIEW)**: 生成完了後、自動的にプレビュータブへ移行します。実際の動作やデザインを確認してください。
4. **調整 (CODE / PREVIEW)**: コードタブで直接手直しするか、プレビュー画面のデバッグパネルからAIへ修正依頼を出します。
5. **公開 (DEPLOY)**: コードタブ右上の「🚀 Deploy」ボタンから、GitHubへプッシュし、Webサイトとして公開します。

## ⚙️ 技術スタック・アーキテクチャ

本アプリケーションは、ポータビリティを最大限に高めるため **Single-File Component（単一ファイル構成）** となっています。

- **Frontend UI**: HTML5, CSS3, Vanilla JavaScript
- **Code Editor**: CodeMirror 6 (ESM import経由で動的読み込み)
- **Local Storage**: IndexedDB (設定、チャット履歴、テーマ設定の永続化)
- **Cryptography**: Web Crypto API
- **APIs**: 
  - Google Generative AI REST API (Gemini)
  - GitHub REST API (インポート、コミット、デプロイ用)
- **Mobile/PWA**: マニフェスト対応、iOS/Androidにおけるセーフエリア(`env(safe-area-inset)`)対応、タッチスワイプUI実装
