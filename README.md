# AI Web Studio 🚀

**AI Web Studio** は、ブラウザ上で完結するAI駆動型のシングルファイルWeb IDE（統合開発環境）です。
AIとのチャットを通じて要件を定義し、コード（HTML/CSS/JS）の生成、リアルタイムプレビュー、直接編集、そして GitHub Pages への直接デプロイまでをシームレスに行うことができます。

モバイル（PWA）にも最適化されており、いつでもどこでもWeb開発が可能です。

## ✨ 主な機能

### 🤖 マルチAIプロバイダー対応
- **Gemini, OpenAI (GPT), Claude, xAI (Grok), Groq** の主要なAIモデルをサポート。
- チャット用とコード生成用で別々のモデルを選択可能。

### 💬 インテリジェントなチャット (CHAT)
- AIとの対話で「作りたいもの」の要件を自動で抽出・リスト化。
- カメラ撮影、ギャラリー、ローカルファイルからの画像読み込み・解析（マルチモーダル対応）。
- AIが会話からユーザーの「ペルソナ（開発の好み）」を学習し、自動更新を提案。
- メッセージ履歴のローカル保存と編集機能。

### 💻 本格的なコードエディタ (CODE)
- **CodeMirror 6** を組み込んだ高性能なコードエディタ。
- シンタックスハイライト、コード検索機能を搭載。
- **GitHub Import**: 既存のGitHubリポジトリからコードを直接読み込み。
- ローカルファイルのアップロード、および生成したコードのダウンロード。

### 👁️ リアルタイムプレビュー (PREVIEW)
- 生成・編集したコードを即座にプレビュー（`iframe` を使用）。
- プレビュー画面から「ここを直して」とデバッグ・修正依頼を直接送信可能。

### 🚀 GitHub Pages へのデプロイ
- ボタン一つで GitHub へのコミットから GitHub Pages の公開までを自動化。
- 新規リポジトリの作成（Public/Private）にも対応。

### 🔒 セキュアでローカルファーストな設計
- **データのローカル保存**: コードやチャット履歴はすべてブラウザの `IndexedDB` に保存。
- **強力な暗号化**: APIキーは `Web Crypto API` (AES-GCM) を用いて暗号化され保存されます。また、保存から2週間で自動消去される安全設計です。

---

## 🛠️ 技術スタック

- **フロントエンド**: HTML5, CSS3, Vanilla JavaScript (フレームワーク不使用のシングルファイル設計)
- **エディタ**: [CodeMirror 6](https://codemirror.net/) (ESM形式で動的インポート)
- **ストレージ**: IndexedDB, LocalStorage
- **セキュリティ**: Web Crypto API
- **外部連携**: GitHub REST API

## 🚀 使い方

本アプリケーションはサーバーサイドの構築を必要とせず、ブラウザのみで動作します。

### 1. 起動方法
リポジトリをクローンまたはダウンロードし、ローカルサーバーで `index.html` を開きます。
※ Web Crypto API（APIキー暗号化）やPWA機能を使用するため、**`localhost` または `https://` 環境での実行が必須**です。

```bash
# Node.jsがインストールされている場合（npxを使用したローカルサーバー起動例）
npx serve .
```

### 2. 初期設定 (SET タブ)
1. 画面右上の **「SET」** タブを開きます。
2. 使用したい **AIプロバイダー** を選択します。
3. 該当する **APIキー** を入力します。（キーは暗号化されブラウザ内にのみ保存されます）
4. プロンプトやコードのカスタマイズルール（ペルソナ）を好みに応じて設定します。
   - GitHubへのデプロイ機能を使用する場合は、**GitHub Token (Classic/Fine-grained)** を設定してください。

### 3. 開発の流れ
1. **CHAT**: 要件をテキストや画像で入力します。AIが提案を行い、仕様リスト（右上の「📋 仕様」）を自動作成します。
2. **GENERATE**: プランが固まると表示される「⚡️ コードを作成」ボタンを押すと、AIがフルスタックのコードを生成します。
3. **PREVIEW**: 生成されたUIと動作を確認します。修正が必要な場合は下部のデバッグパネルから依頼します。
4. **CODE**: 必要に応じてエディタで微調整を行います。
5. **DEPLOY**: CODEタブの「🚀 Deploy」ボタンから、GitHubリポジトリを作成し、GitHub Pagesへ公開します。

## 🎨 テーマとテンプレート機能
- **カスタムテーマ**: SETタブからカラーパレットをカスタマイズし、AIが生成するコードに独自のCSS変数を自動注入できます。
- **テンプレート**: よく使うHTMLのボイラープレートを登録・再利用可能です。

---

## 📱 PWA (Progressive Web App) 対応
モバイルデバイスのブラウザで開き、「ホーム画面に追加」することで、ネイティブアプリのようなフルスクリーン体験が可能です。スワイプ操作による仕様リストの編集など、タッチデバイスに最適化されたUIを提供します。

---

## ⚠️ 注意事項

- 本ツールはクライアントサイド（ブラウザ）から直接外部AIのAPIへリクエストを送信します。開発者コンソール等でネットワークリクエストを監視できる環境ではAPIキーが露出する可能性があるため、共有のPCや信頼できないネットワークでの使用にはご注意ください。
- 一部のAIプロバイダーにおいて、CORS（Cross-Origin Resource Sharing）制限により直接API通信ができない場合は、プロキシサーバー（コード内の `PROXY_WORKER_URL`）を経由するようフォールバック処理が組み込まれています。
